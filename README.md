# Smart-Calculator
## EN
### Description
Smart Calculator - a simple calculator for calculating expressions. Supports writing values to variables. Available operators:
* \+ addition;
* \- subtraction;
* ( ) parenthesies;
* \* multiplication;
* / division;
* ^ exponentiation.
